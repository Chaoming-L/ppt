### 配送单打印及案例分析
### 1️⃣配置模板主要把握*3*个要点 * [打印规格]() 如何选择? * [打印区域]() 如何配置? * [数据字段]() 如何编辑?
主要打印规格: | | [A4](/) | [241纸](/) | [二分纸](/) | | :----:| :---: | :-----: | :----------: | | 模板 | A4 |自定义(210 x 279.4mm)| 二分纸 | | 表单 | A4 |210 x 279.4 mm | 210 x 139.7 mm |
打印区域: * 页眉,页脚: 展示在 [每一页](/) * 签名: 仅展示在 [最后一页](/) * 普通区域(表格): 根据数据量自动适应 * 普通区域(非表格): 占用[固定](/)高度
字段编辑: * `{{}}` 内的文字是[动态数据](/),谨慎编辑 ![预览](./peisong/data.png)

### 列举几个打印异常案例:
打印机入纸不正确,导致偏移👇
![预览](./peisong/jin1.JPG)
解决方案: 调整打印机进纸导轨. ![预览](./peisong/jin2.JPG)

解决方案: 打印规格选择 210 x 279.4 mm ![预览](./peisong/super.JPG)
